What it is

iOS-Hub is a Phone Manager Shortcut (PMS) — a single Apple Shortcut that gives you quick access to phone management tools, built specifically around iOS 26's new capabilities.

Why iOS 26

We started this project on iOS 26 to take advantage of the latest Shortcuts actions. Support for other versions is being explored — check the Download page for what's currently available.

Currently supports iOS 26

Getting started

  • Head to the Download page and pick the card that matches your iOS version
  • Installing the wrong version's shortcut won't work — double-check first
  • Run into an issue? Let us know via the repo's Issues tab
